### En:
#### Task 1
Dynamic Creation and Addition of Items to the Shopping Cart
You need to create a web application for an online store where users can add products to a shopping cart.
When the “Add to Cart” button is clicked for a specific product, a new row should be created in the cart containing information about that product.
To implement this, you can use the following methods and properties:
document.querySelector, createElement, createTextNode, append, textContent, classList.add, and the onclick event.

### Ru:
#### Задача 1

Динамическое создание и добавление элементов в корзину товаров
Вам нужно создать веб-приложение для интернет-магазина, где пользователь сможет добавлять товары в корзину. При нажатии на кнопку «Добавить в корзину» у соответствующего товара должна создаваться новая строка в корзине, содержащая информацию о нём. Для этого можно использовать document.querySelector, createElement, createTextNode, append, textContent, classList.add и событие onclick.
